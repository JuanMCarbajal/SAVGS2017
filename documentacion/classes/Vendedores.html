buscar()

buscar(string  $busqueda) : array

Función que busca vendedores y retorna sus datos.

Esta función recibe un string el cual se compara con los nombres de los vendedores de la tabla vendedores de la base de datos y devuelve los datos de los vendedores cuyos nombres concuerdan parcialmente o en su totalidad con la string recibido.

Parameters

string $busqueda

una string que sirve para realizar la busqueda.

Returns

array —

retorna un array con los datos de la consulta o vacio si no hay resultados.
