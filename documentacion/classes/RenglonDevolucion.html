getRenglonesDevolucion()

getRenglonesDevolucion(string  $codigo_devolucion) : array

Función que busca los articulos devueltos en una devolucion y retorna sus datos.

Esta función recibe un codigo identificador y los compara con los codigos en la tabla renglon_devolucion de la base de datos y devuelve los datos de los articulos devueltos de una devolucion cuyo codigo concuerde en su totalidad con el codigo recibido.

Parameters

string $codigo_devolucion

un string que contiene el codigo identificador

Returns

array —

retorna un array con los datos de la consulta o vacio si no hay resultados.
